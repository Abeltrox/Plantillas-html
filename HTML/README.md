📂 Plantillas Web en HTML

Este repositorio reúne diferentes plantillas web desarrolladas con HTML y CSS. Cada diseño incluye su propio archivo .html, su correspondiente hoja de estilos .css y las imágenes organizadas dentro de carpetas específicas para mantener una estructura clara y ordenada.

La carpeta css contiene todos los archivos de estilos utilizados en el proyecto, mientras que la carpeta image almacena los recursos gráficos. Cada archivo HTML está vinculado a su propio archivo CSS, lo que permite que cada plantilla funcione de manera independiente.

🚀 ¿Cómo visualizar el proyecto?

Para ver el proyecto correctamente, primero descarga los archivos en tu equipo. Luego, abre el archivo index.html en tu navegador web. Es importante no modificar la estructura de las carpetas, ya que esto podría generar errores en la visualización de los estilos o las imágenes.
